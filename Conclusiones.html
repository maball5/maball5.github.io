Ventajas e inconvenientes

Ventajas


  • Utilizar markdown es una forma rápida de crear documentos

  • Uso de una sintaxis sencilla (fácil de aprender) en comparación con HTML

  • Es rápido cambiar el formato del texto (palabras en cursiva, negrita, etc)

Ventajas


  • Es muy sencillo cambiar el tipo de fichero, convertir un .md en un .pdf o en un .epub

  • Reveal.js: se pueden conseguir presentaciones con muy buena estética

  • Fácil de subir y compartir proyectos por Github

Inconvenientes


  • A veces es complicado alinear los elementos usando Markdown

  • No está tan estandarizado como HTML

  • Es necesario tener conocimientos de código sobre todo para crear presentaciones con reveal.js (uso de HTML)
